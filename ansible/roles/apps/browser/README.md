# Browser

Brave keyboard shortcuts can be viewed and changed at:

```text
brave://settings/system/shortcuts
```

Brave content filters can be viewed and changed at:

```text
brave://settings/shields/filters
```

Brave bookmarks can be viewed and managed at:

```text
brave://bookmarks
```

Imported certificates can be managed in Brave at:

```text
brave://certificate-manager/localcerts/usercerts
```

Experimental browser features can be configured at:

```text
brave://flags/
```

- Enable **Smooth Scrolling**.
- Disable **Hardware Media Key Handling**.
